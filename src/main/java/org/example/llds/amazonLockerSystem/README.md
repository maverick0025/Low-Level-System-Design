# Amazon locker system with package delivery with OTP verification

Main features:
- Locker selection flow(based on zip code) : Customer selecting a nearby locker based on delivery address
  - This happens during the product checkout stage on Amazon website.
  - From the options, Customer selects a delivery locker with location may be nearest to them. This booking involves size of the locker recommended or something.
- Delivery flow: Delivery agent comes and deliver the package
- Pickup flow: Customer goes to the locker and collect the package.
